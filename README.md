# GeoMaster: The Ultimate Geography Challenge

A high-performance geography learning game featuring flags, capitals, and competitive modes with real-time feedback and scoring.

## Features

- Flag and capital quiz modes
- Real-time scoring and feedback
- Timed rounds with a live scoreboard

## Prerequisites

- Node.js 18+
- A [Gemini API key](https://aistudio.google.com/apikey)

## Setup

1. Install dependencies:
   ```
   npm install
   ```
2. Copy `.env.example` to `.env.local` and set your `GEMINI_API_KEY`:
   ```
   cp .env.example .env.local
   ```
3. Run the app locally:
   ```
   npm run dev
   ```

## Scripts

- `npm run dev` – start the dev server
- `npm run build` – build for production
- `npm run preview` – preview the production build
- `npm run lint` – type-check with `tsc`
